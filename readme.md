## Setup env

Create .env. 

Get the following from the mentorbot app on slack:
 
```
clientId
clientSecret
clientSigningSecret
```

Add port:
`PORT=3000`

## Start the local server

`yarn start` or `npm run start`

## Authorize with slack

Visit https://localhost:3000/login

Authorize

Start chatting with the bot
